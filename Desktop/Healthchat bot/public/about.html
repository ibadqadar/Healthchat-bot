About the Project

A comprehensive Final Year Project aiming to make basic healthcare information more accessible using Artificial Intelligence.

Core Features

  • AI Chatbot Processes natural language queries to return relevant health FAQs and system capabilities.
  • Nearby Hospitals Uses Geolocation combined with mapping APIs to locate nearby healthcare facilities.
  • Medication OCR Connects to Vision APIs to read text from uploaded medicine images to provide precautions and uses.

Datasets Used

Information in the chatbot and medicine database was curated from credible open sources such as the OpenFDA and WHO open datasets.

Tech Stack

Node.js Express MongoDB Google Gemini AI Google Maps API Vanilla CSS Bootstrap 5
